# Natural Language Processing with Neural Nets

## About this project
A Simple Neural Network for Document Classification using [NLTK](https://www.nltk.org/), [SKLearn](https://scikit-learn.org/stable/) and [PyTorch](https://pytorch.org/)
<!-- Simple Neural Net implementation and training using [PyTorch](https://pytorch.org/) -->

## Technical Notes

### Dependencies
  - `python` version 3.x

### How to run the app
  - Run `pip install -r requirements.txt` in the root folder
  - Run `pip init.py`
  - Run `python server.py`
